# Masin 세이브코드 분류

## 개요
Masin 세이브코드 분류는 게임 세이브 코드 파일을 효율적으로 분류하고 관리할 수 있는 WPF 애플리케이션입니다.

## 주요 기능
- ?? **폴더 자동 로드**: 마지막으로 사용한 폴더를 자동으로 기억하고 로드
- ?? **스마트 분석**: TXT 파일에서 캐릭터 정보와 세이브 코드를 자동 추출
- ?? **캐시 시스템**: 분석 결과를 캐시하여 빠른 재로드 지원
- ?? **클립보드 복사**: 세이브 코드를 클릭 한 번으로 클립보드에 복사
- ?? **상세 정보**: 레벨, 자원, 전투력, 장착 아이템 등 상세 정보 표시
- ?? **사용자 친화적 UI**: 직관적이고 깔끔한 인터페이스
- ?? **실시간 검색**: 캐릭터명으로 빠른 검색 기능
- ?? **캐릭터명 매핑**: charName.json을 통한 마신명 자동 표시

## 시스템 요구사항
- **운영체제**: Windows 10/11 (64-bit)
- **.NET**: .NET 8.0 Runtime (자체 포함 배포 시 불필요)
- **메모리**: 최소 512MB RAM
- **저장공간**: 100MB 이상 여유 공간

## 설치 및 실행
1. Release 페이지에서 최신 버전 다운로드
2. `MasinSaveCodeClassification.exe` 실행
3. 첫 실행 시 세이브 코드 파일이 있는 폴더 선택

## 사용 방법
1. **폴더 선택**: '폴더 변경' 버튼을 클릭하여 세이브 코드 TXT 파일이 있는 폴더 선택
2. **파일 분석**: '파일 분석' 버튼을 클릭하여 세이브 코드 파일 분석
3. **캐릭터 검색**: 검색창에 캐릭터명을 입력하여 빠른 검색
4. **캐릭터 선택**: 좌측 캐릭터 목록에서 원하는 캐릭터 선택
5. **세이브 코드 복사**: 우측에서 원하는 세이브 코드를 선택하고 '코드 복사' 버튼 클릭

## 캐릭터명 매핑 기능
### charName.json 파일
```json
[
  {
    "마신": "갈증",
    "이름": ["새로운 피의 주인", "피로 역사를 다시 쓰는 자"]
  },
  {
    "마신": "듀크",
    "이름": ["최초로 종족을 초월한 자", "광활히 세상을 드넓히는 자"]
  },
  {
    "마신": "죽천",
    "이름": ["죽음을 다스리는 자", "죽음을 구원하는 자"]
  }
]
```

- **기능**: 원본 캐릭터명("이름" 배열의 값들)이 JSON의 "이름" 배열과 일치하면 "마신" 값으로 UI에 표시
- **다중 이름 지원**: 하나의 마신에 여러 개의 이름을 배열로 매핑 가능
- **위치**: 실행 파일과 같은 폴더에 `charName.json` 파일 배치
- **동작**: 매핑되지 않은 캐릭터는 원본 이름으로 표시
- **검색**: 마신명과 원본명 모두로 검색 가능
- **형식 지원**: 
  - 배열 형태: `["이름1", "이름2"]`
  - 단일 문자열: `"이름"` (기존 호환성)
  - 쉼표 구분: `"이름1, 이름2"` (자동 분할)

## 기술 스택
- **Framework**: .NET 8.0, WPF
- **언어**: C# 12.0
- **패키지**: 
  - HandyControl 3.5.1 (UI 컴포넌트)
  - TextCopy 6.2.1 (클립보드 관리)

## 파일 형식 지원
애플리케이션은 다음 형식의 TXT 파일을 지원합니다:
```
call Preload( "캐릭터: 캐릭터명" )
call Preload( "Code: ABC-DEF-GHI" )
call Preload( "레벨: 50" )
call Preload( "금: 10000" )
call Preload( "나무: 5000" )
call Preload( "무력: 100" )
call Preload( "요력: 80" )
call Preload( "영력: 90" )
call Preload( "경험치: 1500000" )
call Preload( "아이템1: '무기명'" )
...
```

## 데이터 저장 위치
- **설정 파일**: `%APPDATA%\SaveCodeClassification\config.json`
- **캐시 파일**: `%APPDATA%\SaveCodeClassification\analysis_cache.json`
- **매핑 파일**: `charName.json` (실행 파일과 같은 위치)
