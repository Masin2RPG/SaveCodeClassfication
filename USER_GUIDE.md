## 프로그램 소개

**Masin 세이브코드 분류**는 게임 세이브 코드 파일을 효율적으로 관리할 수 있는 데스크톱 애플리케이션입니다.

### 주요 특징
- ?? **빠른 분석**: 대량의 세이브 코드 파일을 빠르게 분석
- ?? **스마트 분류**: 캐릭터별로 자동 분류 및 정리
- ?? **캐시 시스템**: 한 번 분석한 결과를 저장하여 다음 실행 시 빠른 로드
- ?? **간편 복사**: 클릭 한 번으로 세이브 코드를 클립보드에 복사
- ?? **상세 정보**: 캐릭터의 레벨, 자원, 전투력, 장착 아이템 등 상세 정보 제공

## 설치 방법

### 시스템 요구사항
- **운영체제**: Windows 10 또는 Windows 11 (64-bit)
- **메모리**: 최소 512MB RAM
- **저장공간**: 100MB 이상 여유 공간
- **기타**: .NET Runtime 불필요 (자체 포함)

### 설치 과정
1. 배포 파일 다운로드
2. `MasinSaveCodeClassification.exe` 파일을 원하는 위치에 저장
3. 더블클릭으로 실행

> ?? **팁**: 바탕화면이나 프로그램 폴더에 바로가기를 만들어 두면 편리합니다.

## 첫 실행 및 설정

### 1. 첫 실행
프로그램을 처음 실행하면 다음과 같은 화면이 나타납니다:

```
┌─────────────────────────────────────────────────┐
│ ?? Masin 세이브코드 분류                        │
├─────────────────────────────────────────────────┤
│ 폴더 설정                                       │
│ [폴더 변경] [새로고침]                          │
│ 저장된 폴더를 자동으로 로드합니다...            │
│                                                 │
│ ?? 앱 시작 시 자동으로 폴더를 로드합니다        │
└─────────────────────────────────────────────────┘
```

### 2. 폴더 선택
1. **"폴더 변경"** 버튼 클릭
2. 세이브 코드 TXT 파일이 있는 폴더 선택
3. **"확인"** 클릭

### 3. 자동 저장
선택한 폴더는 자동으로 저장되어 다음 실행 시 자동으로 로드됩니다.

## 기본 사용법

### 1. 파일 분석
폴더를 선택한 후 **"파일 분석"** 버튼을 클릭합니다.

```
분석 과정:
1. TXT 파일 검색 중...
2. 파일 분석 중... (1/50)
3. 캐릭터 분류 중...
4. 분석 완료!
```

### 2. 캐릭터 선택
좌측 **캐릭터 목록**에서 원하는 캐릭터를 클릭합니다.

### 3. 세이브 코드 확인
우측 **세이브 코드 목록**에서 원하는 세이브 코드를 선택합니다.

### 4. 코드 복사
**"코드 복사"** 버튼을 클릭하여 클립보드에 복사합니다.

## 고급 기능

### 캐시 시스템
- **자동 캐시**: 분석 결과가 자동으로 저장됩니다
- **빠른 로드**: 다음 실행 시 캐시된 결과를 빠르게 로드
- **캐시 삭제**: 필요 시 **"캐시 삭제"** 버튼으로 캐시 초기화

### 상세 정보 보기
세이브 코드를 선택하면 하단에 다음 정보가 표시됩니다:
- ?? **기본 정보**: 레벨, 경험치
- ?? **자원 정보**: 금, 나무
- ?? **전투 능력치**: 무력, 요력, 영력
- ?? **장착 아이템**: 아이템1~6

### 파일 새로고침
- **"새로고침"** 버튼: 폴더의 새로운 파일 확인
- **자동 감지**: 파일 변경 시 자동으로 업데이트

## 문제 해결

### 파일이 인식되지 않을 때
1. **파일 형식 확인**: TXT 파일인지 확인
2. **파일 내용 확인**: 다음 형식이 포함되어 있는지 확인
   ```
   call Preload( "캐릭터: 캐릭터명" )
   call Preload( "Code: ABC-DEF-GHI" )
   ```

### 프로그램이 느릴 때
1. **캐시 삭제**: 손상된 캐시가 원인일 수 있음
2. **파일 수 확인**: 너무 많은 파일이 있는지 확인
3. **컴퓨터 재시작**: 메모리 정리

### 복사가 안 될 때
1. **다른 프로그램 확인**: 클립보드를 사용하는 다른 프로그램 종료
2. **상세 창 사용**: 복사 실패 시 나타나는 상세 창에서 수동 복사

## 자주 묻는 질문

### Q: 지원하는 파일 형식은?
A: `call Preload()` 형식의 TXT 파일을 지원합니다.

### Q: 한 번에 몇 개까지 파일을 처리할 수 있나요?
A: 이론적으로 제한이 없지만, 수천 개 이상의 파일은 처리 시간이 오래 걸릴 수 있습니다.

### Q: 설정 파일은 어디에 저장되나요?
A: `%APPDATA%\SaveCodeClassification\` 폴더에 저장됩니다.

### Q: 프로그램을 삭제하려면?
A: 실행 파일을 삭제하고, 필요 시 설정 폴더(`%APPDATA%\SaveCodeClassification\`)도 삭제하세요.

### Q: 업데이트는 어떻게 하나요?
A: 새 버전을 다운로드하여 기존 파일을 덮어쓰면 됩니다. 설정은 자동으로 유지됩니다.

### Q: 다른 컴퓨터에서도 사용할 수 있나요?
A: 네, 실행 파일만 복사하면 어디서든 사용 가능합니다.
