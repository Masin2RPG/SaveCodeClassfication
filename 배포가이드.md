# Masin 세이브코드 분류 도구 - 배포 가이드

## 배포 완료 정보

### 생성된 배포 파일
- **파일명**: SaveCodeClassfication.exe
- **크기**: 약 197 MB (206,937,154 bytes)
- **배포 유형**: Self-Contained Single File
- **대상 플랫폼**: Windows x64
- **.NET 버전**: .NET 8.0

### 배포 특징
- ? **Self-Contained**: .NET 런타임이 포함되어 별도 설치 불필요
- ? **Single File**: 모든 의존성이 하나의 exe 파일로 통합
- ? **ReadyToRun**: 최적화된 네이티브 코드로 빠른 시작
- ? **윈도우 전용**: Windows 10/11에서 실행 가능

## 시스템 요구사항

### 최소 요구사항
- **운영체제**: Windows 10 1809 이상 또는 Windows 11
- **아키텍처**: x64 (64-bit)
- **메모리**: 최소 2GB RAM 권장
- **저장공간**: 최소 300MB 여유 공간

- **데이터베이스**: MySQL 서버 (연결 설정 필요)

## 설치 및 실행

### 설치 방법
1. `SaveCodeClassfication.exe` 파일을 원하는 폴더에 복사
2. 바로 실행 가능 (추가 설치 불필요)

### 실행 방법
1. `SaveCodeClassfication.exe` 더블클릭으로 실행
2. 또는 명령줄에서 실행:
   ```
   SaveCodeClassfication.exe
   ```

### 첫 실행 시 설정
1. 데이터베이스 연결 정보 입력
2. 사용자 계정 로그인 또는 회원가입
3. 관리자 계정으로 토큰 생성 (필요시)

## 문제 해결

### 일반적인 문제
1. **실행되지 않는 경우**
   - Windows Defender 또는 백신 프로그램 확인
   - 관리자 권한으로 실행 시도
   - 파일 손상 여부 확인

2. **데이터베이스 연결 오류**
   - MySQL 서버 실행 상태 확인
   - 연결 정보 (서버, 포트, 사용자명, 비밀번호) 재확인
   - 방화벽 설정 확인

3. **성능 문제**
   - 충분한 메모리 확보
   - 백그라운드 프로그램 종료
   - 디스크 여유 공간 확보

### 로그 파일
- 문제 발생 시 이벤트 뷰어 또는 애플리케이션 로그 확인
- Visual Studio Output 창에서 디버그 정보 확인 가능
