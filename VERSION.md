#### 기술적 구현
- .NET 8.0 WPF 기반 데스크톱 애플리케이션
- 단일 파일 자체 포함 배포
- JSON 기반 설정 및 캐시 시스템
- 정규식 기반 스마트 파싱
- 비동기 파일 처리

#### 지원 파일 형식
- call Preload() 형식의 TXT 파일
- 캐릭터 정보, 세이브 코드, 스탯, 아이템 정보 자동 추출

#### 시스템 요구사항
- Windows 10/11 (64-bit)
- 최소 512MB RAM
- 100MB 이상 저장공간
